README FOR REVIEWERS
All files relevant for reviewers have been compiled in the directory called FOR_REVIEWERS for ease of access. 

The files are:

- 00.README_FOR_REVIEWERS: this file
- 18S_rRNA.fa: 18S rDNA sequence from Metilla boricua
- 18S_tree.aln.gb: alignment file for 18S phylogenetic tree construction
- 18S_tree.aln-gb.raxml.support: ML tree 
- 28s.aln: alignment file for 28S phylogenetic tree construction
- 28s.alm-gb: alignment file with inunformative sites removed using G-blocks
- 28s.aln-gb.raxml.support: ML tree
- 28S_rRNA.fa: 28S rDNA sequence from Metilla boricua
- Final_18S_nums.xlsx: file with all accession mnumbers for all 18S sequences used for alignment
- Final_28S_nums.xlsx: file with all accession mnumbers for all 28S sequences used for alignment
- Metilla_boricua_mtDNA.flatfile: flatfile with annotate mtGenome of Metilla boricua
- X_deweertae_mtDNA.flatfile: flatfile with annotate mtGenome of Xestosongia deweerdtae

# Mystery-sponge
This repository contains sequence data for the three sponges inside Haliclona plakophila/Plakortis symbiotica association. 
The directory structure is:
- README.md: this file           
- documents: proposal doc; place the ms. here?          
- mtDNA: assemblies and analysis of mtDNA from the three species
- phylogeny: phylogenetic analysis based on mtDNA and rRNA sequences

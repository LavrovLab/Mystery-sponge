# Mystery-sponge
This repository contains sequence data for the three sponges inside Haliclona plakophila/Plakortis symbiotica association. 
The directory structure is:
- Haliclona-cladeB: data for Haliclona in clade B     
- Haplosclerida-cladeD: data for the mystery sponge 
- Plakortis: mt data for Plakortis symbiotica            
- README.md: this file           
- documents: proposal doc; place the ms. here?          
- rRNA: Momin's analysis of rRNA sequences

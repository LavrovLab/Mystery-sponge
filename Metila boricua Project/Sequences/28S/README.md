# Sequences: 28S
This folder contains genomic rDNA sequences of the large subunit generated and used for this study. The directory structure is as follows:


- README.md: this file
- Hplakophila_28S.txt: Data for Haliclona plakophila
- Mboricua_28S.txt: Data for the new species Metilla boricua
- 28S_dataset.csv: spreadsheet containing all accession mumbers for 28S data used for phylogenetic analysis in this study.

### mtDNA
This repository contains mtDNA data for PLakortis symbiotica and its three known sponge symbionts, Halliclona plakophila, Xestospongia deweerdtae, and Metilla boricua. The directory structure is:

README.md: this file
H_plakophila: data for Haliclona plakohila
Metilla_boricua: data for the novel species Metilla boricua
P_symbiotica: data for Plakortis symbiotica
X_deweerdtae: Data for Xestospongia deweerdtae

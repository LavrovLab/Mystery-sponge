# Metilla boricua Project

This repository contains all data relecant to the identification and descrition of the novel genus et species, Metilla boricua. The directory structure is: 

- README.md: this file
- Correspodance_Analysis: data and R scripts used to perform correspondance analyses
- Documents: proposal documents, pre-print manuscript, figures and images
- Phylogenetics: alignments, and phylogenetic trees 
- Dequences: genomic sequences generated for this study

# README

This folder contain the files used for correspondence analyses.

The contents are as follows:

- README: This file
- Clades.xlsx: spreadsheet delimiting species by their clades 
- Haplo_aa.aln: MSA of concatinated ammino acid sequecnces infered from haploscerlida mtDNA
- Haplo_complete_Mt.fasta: unaligned complete mtDNA of different haplosclerida species, starting at rnl genes
- HplakGenes.txt: individual DNA sequences of mitochondrial genes in Haliclona plakophila
- MboricuaGenes.txt: individual DNA sequences of mitochondrial genes in Metilla boricua
- Mboricua_cox2.txt: the cox2 gene of Matilla boricua 
- XdGenes.txt: individual DNA sequences of mitochondrial genes in Xestospongia deweerdtae
- codon_corres.R: the R script used 
- Hplak_cox2.txt: the cox2 gene of Haliclona plakophila
- Xd_cox2.txt: the cox2 gene of Xestosponfia deweerdtae
